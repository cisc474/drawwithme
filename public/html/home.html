Introduction to Gameplay

How to play:

  • Create a display name!
  • Join a game!
  • If selected as the drawer, click "start game" and draw a picture of the word you're given!
  • If not the drawer, try and guess the word by typing it into the chat box!
  • Don't be afraid to chat and have fun using the chat box during the game!
